# **Ncurses Maze Game**

## Installation (Linux)
- Make sure you have ncurses installed
- Clone repo
- Inside repo folder run 
```
g++ src/*.cpp -lncurses
```
- Choose map while running, for example: 
```
./a.out map2.txt
```
---

### How to play
- Use arrow keys to move H (Human)
- Find the Sword (S) before Orc (O) does
